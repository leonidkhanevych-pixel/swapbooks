# SwapBooks

SwapBooks — це онлайн-платформа для безкоштовного обміну книгами між користувачами. Мета проєкту — зробити обмін літературою простим, доступним і популярним способом поширення знань.

## Мета

- Створити зручний сервіс для пошуку й обміну книгами.
- Формувати спільноту активних читачів.
- Популяризувати культуру читання та повторного використання книжок.
- Підтримувати студентів та учнів через обмін літературою.

## Основні функції

- Перегляд доступних книг у каталозі.
- Додавання нових книг через форму.
- Контакти та зворотний зв’язок із адміністрацією.
- Адаптивний дизайн для мобільних пристроїв, планшетів та ПК.
- Базова клієнтська валідація форм.

## Технології

- HTML5
- CSS3 (Flexbox, Grid, адаптивний дизайн)

## Примітки

- Усі форми мають базову HTML5-валидацію.
- Меню навігації адаптивне і підтримує мобільну версію з бургер-меню.
- Кросбраузерна перевірка проводилась у Chrome, Firefox, Edge та Safari.
