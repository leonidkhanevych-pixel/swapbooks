# SwapBooks

SwapBooks — онлайн-платформа для безкоштовного обміну книгами між користувачами.

## Мета

- Створити зручний сервіс для пошуку й обміну книгами.
- Формувати спільноту активних читачів.
- Популяризувати культуру читання та повторного використання книжок.

## Основні функції

- Перегляд каталогу книг.
- Додавання нових книг через форму.
- Контакти та зворотний зв’язок із адміністрацією.
- Адаптивний дизайн для ПК, планшетів та мобільних пристроїв.
- Клієнтська валідація форм (ім’я, email, повідомлення).
- Динамічна дата у футері.
- Акордеон "Показати більше".
- Зміна теми сайту (світла/темна).
- Підсвітка меню при наведенні миші.
- Зміна розміру шрифту за допомогою клавіш ArrowUp/ArrowDown.

## Технології

- HTML5
- CSS3
- JavaScript 

## Примітки

- Меню адаптивне з бургер-меню для мобільних пристроїв.
- Кнопки стилізовані та помітні на обох темах.
- Кросбраузерна перевірка проводилась у Chrome, Firefox, Edge та Safari.
